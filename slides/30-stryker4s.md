## Mutation switching in practice

```scala " data-line-numbers=true
def greaterThan(a: Int, b: Int): Boolean = {
  sys.env("ACTIVE_MUTATION") match {
    case "0" => a >= b // Mutant 1
    case "1" => a < b  // Mutant 2
    case "2" => a == b // Mutant 3
    case _   => a > b  // Original mutation
  }
}
```
